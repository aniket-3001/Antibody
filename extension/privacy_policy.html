Antibody — Privacy Policy

Last updated: July 2025

What the extension does

Antibody is a browser extension that lets you check suspicious text (messages, emails, web-page snippets) against the Antibody scam-pattern graph. It calls the read-only /scan endpoint of the Antibody API.

Data we collect

When you click Check it or use the right-click menu, the selected text is sent over HTTPS to the Antibody API at https://antibody-251148844884.asia-south1.run.app. The /scan endpoint is read-only: it returns a verdict and does not store your text, does not create a report, and does not record your IP address or any identifiers in its database.

We do not collect, store, or share any personal data.

Permissions used

  • activeTab / scripting — to read the visible text of the current page when you click "Scan this page".
  • contextMenus — to add the right-click "Check … with Antibody" menu item.
  • notifications — to show a desktop alert with the verdict after a right-click check.
  • host_permissions to the Antibody API URL — so the background service worker can fetch the API without being blocked by CORS.

Third parties

The extension calls only the Antibody API. The API itself may internally call an NVIDIA NIM or OpenAI-compatible LLM to generate the explanation text, but your raw message is not retained by those services beyond the single API call.

Children

This extension is not directed at children under 13.

Changes

If we change this policy we will update the date at the top. Continued use of the extension after a change constitutes acceptance.
